# Genspark Master 학습 플랫폼 초기 구축

## 작업 일자
2026-04-04

## 작업 개요
Genspark AI 학습 플랫폼을 Claude Master 프로젝트의 디자인 컨셉을 기반으로 신규 구축했습니다. 동일한 기술 스택(React 19 + Vite 6)을 사용하되, Genspark 브랜드 컬러(보라색 #7C3AED)와 Genspark AI의 학습 콘텐츠로 구성했습니다.

## 디자인 시스템

### Claude Master와의 차이점

| 항목 | Claude Master | Genspark Master |
|------|-------------|----------------|
| 기본 컬러 | #1B3A6B (다크블루) | #7C3AED (보라색) |
| 액센트 컬러 | #D97706 (앰버) | #2563EB (일렉트릭블루) |
| 히어로 그라데이션 | 블루 계열 | 퍼플 계열 |
| 인증 | Supabase Auth (Google/Kakao) | 없음 (학습 전용) |
| 커뮤니티 | 게시판 | 없음 |
| DB | Supabase PostgreSQL | 없음 (정적 사이트) |

### 공통 디자인 요소

- 글래스모피즘 네비게이션 바
- 다크/라이트/오토 모드
- 5가지 컬러 테마
- 한국어/영어 다국어 지원
- 반응형 (5개 브레이크포인트)
- 사이드바 목차 가이드 레이아웃
- 코드 블록 + 신택스 하이라이팅
- TipBox 컴포넌트
- 모바일 최적화 (가로 스크롤 탭)
